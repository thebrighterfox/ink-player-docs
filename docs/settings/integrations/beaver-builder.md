# Beaver Builder

Beaver Builder is a page builder for WordPress. Enabling this integration adds a native **Ink Player module** to Beaver Builder, so you can place a player anywhere on a page using the visual builder.

## What this integration does

- Adds an Ink Player module to Beaver Builder.
- Lets you drop the module onto a page and pick the [media](/media) item or [playlist](/playlists) it renders.

<div class="ink-timeline">

## Step 1 — Enable the integration

This integration is **enabled by default** — when Beaver Builder is active, the module is available with no setup. To confirm or toggle it:

1. Make sure the **Beaver Builder** plugin is installed and active on your site.
2. Go to **Settings → [Integrations](/settings/integrations)**.
3. Find the **Beaver Builder** card under the **Builder** category and confirm the **Enable** toggle is on.

## Step 2 — Add the Ink Player module

Edit a page with Beaver Builder, then drag the **Ink Player** module onto your layout.

## Step 3 — Choose your content

In the module settings, use the picker to select the [media](/media) item or [playlist](/playlists) to display, and adjust the module options.

  <img src="/screenshots/settings/integrations/beaver/block-options.png" alt="The Ink Player Beaver Builder module options">
  <figcaption>Pick the media or playlist in the module options</figcaption>

## Step 4 — Preview and publish

Preview the player live in the editor, then publish or save the page.

::: tip
You can use the same module for a single media item or a full [playlist](/playlists) — just choose accordingly in the picker.
:::

## Requirements

- The Beaver Builder plugin must be installed and active.
- At least one Ink Player [media](/media) item or [playlist](/playlists) to render.

## Notes

- If the Ink Player module doesn't appear in Beaver Builder, confirm both Beaver Builder and this integration are enabled.
