# Dashboard

The **Dashboard** is the first screen inside the Ink Player admin area. It gives you a compact overview of your media library, recent activity, and the next actions most site owners need.

## What you see

The dashboard is designed for quick scanning:

- **Performance summary** — recent views, viewers, watch time, and engagement signals when [Analytics](/analytics) is enabled.
- **Top content** — media items that are performing best during the current reporting window.
- **Recent media** — the newest video and audio items added to the library.
- **Recent playlists** — playlist shortcuts so you can keep collections current.
- **Quick actions** — links to add media, create playlists, manage integrations, and review settings.
- **Resources** — useful documentation and support links.

::: info
Analytics cards only become meaningful after you enable tracking in [Settings → Analytics](/settings/analytics). Until then, use the dashboard as a launch point for setup and library management.
:::

## Recommended first actions

1. Add your first item in [Media](/media).
2. Choose a default look in [Settings → Presets](/settings/presets).
3. Connect providers in [Settings → Integrations](/settings/integrations) if you use Bunny.net, Mux, email tools, or page builders.
4. Turn on [Analytics](/settings/analytics) when you are ready to collect playback data.
5. Add [Overlays](/guide/overlays) or [Ads](/settings/ads) where they support your content goals.

## When to use another page

Use the dashboard for orientation and shortcuts. Use the dedicated sections for focused work:

| Need | Go to |
| --- | --- |
| Add, edit, duplicate, or protect a media item | [Media](/media) |
| Build an ordered video or audio collection | [Playlists](/playlists) |
| Review playback trends and audience behavior | [Analytics](/analytics) |
| Export or route captured emails | [Leads](/leads) |
| Change global defaults, providers, ads, or webhooks | [Settings](/settings) |
