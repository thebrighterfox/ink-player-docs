# Installation

Ink Player installs like any standard WordPress plugin. This page covers the requirements, the install steps, and where the plugin appears once it is active.

## Requirements

Before installing, make sure your site meets these minimums:

- **WordPress** 6.0 or later
- **PHP** 8.0 or later

::: warning
If your host runs an older version of PHP or WordPress, update before activating Ink Player. The plugin will not run correctly on unsupported versions.
:::

## Install from a zip file

1. Download the Ink Player plugin zip file.
2. In the WordPress admin, go to **Plugins → Add New**.
3. Click **Upload Plugin** at the top of the page.
4. Choose the Ink Player zip file and click **Install Now**.
5. When the upload finishes, click **Activate Plugin**.

## Install by uploading to the server

If you prefer to upload manually:

1. Unzip the plugin file.
2. Upload the resulting folder to `wp-content/plugins/` on your server.
3. In the WordPress admin, go to **Plugins**, find **Ink Player**, and click **Activate**.

## Where it appears

After activation, a top-level **Ink Player** menu appears in the WordPress admin sidebar. From there you can reach:

- **Dashboard** — your overview with KPIs and recent activity
- **Library** — [Media](/media) and [Playlists](/playlists)
- **System** — [Analytics](/analytics), [Leads](/leads), and [Settings](/settings)

::: tip
A floating **Docs** tab on the right edge of every Ink Player screen opens the documentation for whatever you are looking at.
:::
